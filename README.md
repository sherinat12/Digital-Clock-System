# 🕒 Digital Clock System 

## 📌 Description
This project is developed using the C programming language.  
It is a Digital Clock System that displays the current time continuously in the terminal.

---

## 🎯 Features
- Live digital clock
- Real-time updates every second
- Colored console interface
- Simple and attractive UI
  
---

## 🛠️ Technologies Used
- C Programming Language
- GCC Compiler
- VS Code / CodeBlocks

---

## ▶️ How to Run

### Compile
gcc digital_clock.c -o clock

### Run
./clock

(For Windows)
clock.exe
